  • Relative Position: taking an element and moving it from one place where it naturally sit on the page to another place just slightly off.
  • Absolute positioning lets us move an element based on its parent but the trick here is that the parent element have to be position relative.
    Top right
  • Fixed positioning-sticks an elements to a certain part of the viewport.
